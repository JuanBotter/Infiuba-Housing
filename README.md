# Infiuba Alojamientos Web

[Infiuba housing](https://infiuba-housing.vercel.app/) helps exchange students find and review accommodations in Buenos Aires.

## Overview

- Multilingual UI (English, Spanish, French, German, Portuguese, Italian, Norwegian).
- Map-first browsing with filters and review highlights.
- Role-based access for reviews and contact details.
